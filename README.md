# JHGestureBlock
Use gesture with block，手势，Block

---

# Usage
```
    [self.view addGestureRecognizer:JHGestureType_Tap block:^(UIView *view, UIGestureRecognizer *gesture) {
        // do your stuff!
        // 是时候露一手了！
        
        // example
        [view removeGestureRecognizer:gesture];
    }];
```
